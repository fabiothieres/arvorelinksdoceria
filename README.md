# 🧁 Doceria Floripa - Link in Bio

Uma página de links rápida, elegante e totalmente responsiva desenvolvida para a **Doceria Floripa**. 

Este projeto funciona como uma "Árvore de Links" (Linktree), ideal para ser utilizada na biografia do Instagram ou TikTok da doceria. O objetivo principal é centralizar os canais de comunicação, permitindo que os clientes acessem o site oficial ou façam encomendas diretamente pelo WhatsApp com apenas um clique.

## ✨ Funcionalidades

- **Design Responsivo:** Layout adaptável para qualquer tamanho de tela (Mobile e Desktop).
- **Redirecionamento Inteligente:** Botão de WhatsApp configurado com uma mensagem padrão para facilitar a vida do cliente na hora da encomenda.
- **Performance Otimizada:** Importação de fontes enxuta para garantir um carregamento quase instantâneo, vital para conexões móveis.
- **UI/UX Focada no Nicho:** Paleta de cores escolhida estrategicamente para remeter ao universo da confeitaria e chocolates.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica da página.
- **CSS3:** Estilização, layout via Flexbox e Media Queries para responsividade.
